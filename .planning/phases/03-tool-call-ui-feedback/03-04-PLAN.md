       - Click again to collapse.
       - Request JSON shows input object without any `apiKey` field visible (redacted to `[REDACTED]` if present).

       **TOOL-05 (vertical stack):**
       - If the question triggers multiple tools (search_law then get_law_text), the chips stack vertically, each on its own line, no bullet points, no numbering, no border around the group.
       - Chip block appears ABOVE the answer text, not below.

       - The old `검색 중...` text NEVER appears.
       - Immediately after sending, a 3-bar skeleton bubble appears in the assistant avatar column.
       - Once the first tool chip arrives, the skeleton disappears.
